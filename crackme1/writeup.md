# CrackMe 1

## Nama File
Secret_key.exe

## Tujuan
Menganalisis file menggunakan Ghidra.

## Langkah

1. Membuat project Ghidra.
2. Mengimpor file.
3. Menjalankan Auto Analysis.
4. Melihat Defined Strings.
5. Membuka fungsi main().

## Hasil Analisis

- Program meminta input dari pengguna.
- Terdapat proses validasi password.
- Ditemukan beberapa string seperti "Correct" dan "Wrong".
- Fungsi utama berada pada main().

## Kesimpulan

Program melakukan pemeriksaan password sebelum menampilkan hasil.
